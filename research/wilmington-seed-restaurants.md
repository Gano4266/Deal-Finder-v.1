# Wilmington Seed Restaurants

This seed list defines the first Wilmington-area restaurants to review for a static-data `/tonight` MVP. It is an operating backlog, not a source of verified public deal claims.

Exact deals from user notes, screenshots, aggregators, comments, or social chatter remain `needs_review` until supported by an acceptable official source or direct confirmation.

Source status values:

- `verified`: acceptable source evidence exists in the repo.
- `likely`: restaurant appears appropriate for the seed set, but source evidence still needs capture.
- `needs_review`: source, location, recurring-special status, or exact deal terms need human verification.

Source priority values:

- official website
- official social
- direct confirmation
- third-party discovery
- user note
