# Future Workers

This folder is a placeholder for future scanner and validation workers.

## Planned Stages

1. Planner: choose sources due for review.
2. Fetcher: retrieve or browser-render source pages.
3. Capture store: save text, metadata, hash, and screenshots when useful.
4. Extractor: use AI to create deal candidates.
5. Validator: enforce source, location, freshness, and copy gates.
6. Deduper: merge repeated or unchanged deals.
7. Publisher: promote reviewed deals and expire stale records.
8. Auditor: summarize changes and review backlog.

Workers should start batch-oriented. Avoid high-frequency crawling until the first 25-restaurant workflow is reliable.
